# Home components

## Metricas da homepage
- O bloco de metricas deve manter o layout original do design (linha simples, sem card extra).
- Para os numeros animados, use `@number-flow/react`.
- Estado inicial deve ser `0` e a animacao acontece quando os dados da API chegam.
- Neste bloco especifico, nao usar Suspense/skeleton para loading.
