# Devroast

Devroast e um app web que recebe codigo e devolve uma analise direta, com um toque de humor. A experiencia e focada em modo escuro, tipografia monoespacada e feedback rapido.

## O que voce encontra
- Colagem de codigo com visual de editor.
- Modo de analise com tom “roast”.
- Leaderboard com os codigos mais “questionaveis”.

## Evento
Este projeto foi construido durante o evento **NLW** da Rocketseat, nas aulas do evento.

## Status
Versao inicial com dados estaticos (sem API).
